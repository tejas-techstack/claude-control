# Divergence moves — engineer novelty, then verify it

Original means checked, not just imagined. You can't deviate from a norm you haven't named, and you can't claim novelty you haven't searched. This is the toolkit.

## Step 1 — name the genre defaults

Quick `/github-explore` + web scan of the idea space. Write down what *every* example shares — the conventions, the default UX, the assumed platform, the standard business model. Those shared assumptions are your raw material.

## Step 2 — apply divergence moves to the defaults

Generate 6-8 raw spins fast; don't self-censor yet.

- **Invert an assumption** — what if it worked offline / backwards / for the opposite user / without the "core" feature?
- **Transfer a mechanic from a foreign domain** — git for recipes, ELO ranking for chores, speedrun timers for studying, spaced-repetition for code review.
- **Change the binding constraint** — one button only, zero server, must run on a 2010 laptop, must work with no account, 100% keyboard.
- **Merge two things nobody merges** — a linter + a journaling app; a package manager + a museum.
- **Subtract the sacred feature** — remove the thing every competitor treats as mandatory (the feed, the login, the dashboard).
- **Extremify a dimension** — 10x faster, 100x cheaper, 1000x smaller scope, or absurdly over-specialized for one niche.
- **Change who it's for** — the same tool re-aimed at kids / experts / one specific profession.
- **Flip the interaction model** — CLI → ambient, app → email, dashboard → single number.

## Step 3 — verify novelty (mandatory)

Search the 3 most promising spins *specifically*. Finding prior art is fine and expected — state how the spin differs, or iterate again. **Never claim "no one has done this" without having searched**; report what the search actually showed (with links).

## Step 4 — reality-check the winner

- Feasible with free tools?
- Is there a weekend-sized wedge — the smallest version that proves the idea?
- What's the honest hard part (the thing that makes it *not* trivial)?

## Output

Top 3 spins, each: the spin in one sentence → why the genre never does this → nearest existing thing found (link) and the difference → feasibility note. Then: **"My pick + the 30-minute first experiment."** Offer `/cross-check` to spec it, `/research` for academic prior art.
